## Rocky Linux 9.7 发布说明

您可以在 [此处的上游发布说明](https://docs.redhat.com/en/documentation/red_hat_enterprise_linux/9/html/9.7_release_notes/index) 中找到大多数变更的完整清单。

## 系统升级

您可以通过执行 `sudo dnf -y upgrade` 来完成从 Rocky Linux 9.x 到 Rocky Linux 9.7 的升级。

!!! Note "说明"

    Rocky Linux 不提供任何 Rocky Linux 8 版本的升级途径。我们建议进行全新的操作系统安装以升级到 Rocky Linux 9.7。

## 镜像

提供除标准安装镜像外的其他一些镜像，包含云和容器平台的镜像。

您可以在 [SIG/Cloud 维基页面](https://sig-cloud.rocky.page/) 上找到有关 Cloud Special Interest Group 制作镜像以及如何参与其中的更多信息。

## 安装

安装之前，[请按照此步骤确保您的 CPU 兼容性！](https://docs.rockylinux.org/gemstones/test_cpu_compat/)

要安装 Rocky Linux 9.7，请转到 [下载页面](https://rockylinux.org/download/) 并下载与您架构相对应的版本。

### 安全性

以下是最新发布的 Rocky Linux 9.7 中与安全相关的亮点。 有关安全相关更改的完整清单，请参阅 [此处的上游链接](https://docs.redhat.com/en/documentation/red_hat_enterprise_linux/9/html/9.7_release_notes/new-features#new-features-security)：

- 系统范围加密策略：您可以通过新的 PQ 子策略启用 PQC（Post-Quantum Cryptography，后量子密码学）支持。 许多应用程序和工具已针对 PQC 中的相应算法进行了新增和改进。

### 动态编程语言、web和数据库服务器

- Node.js 24
- Valkey 8

### 编译器和开发工具

#### 系统工具链组件

- Glibc 2.34
- Annobin 12.98

#### 性能工具和调试器

- GDB 16.3
- Valgrind 3.25.1
- SystemTap 5.3
- Dyninst 13.0.0
- elfutils 0.193
- libabigail 2.8

#### 编译器工具集

- GCC Toolset 15 包括：

  - GCC 15.1
  - Binutils 2.44

- LLVM Toolset 20.1.8

- Rust Toolset 1.88.0

- Go Toolset 1.24

- .NET 10.0

### RL 网页控制台（Cockpit）

Cockpit 是一个基于网页的界面，用于管理和监控您的本地系统。

这个新版本带来了基于 PatternFly 6 改进的 UI 风格。

#### Image Builder 工具

‌Image Builder 工具现在能够创建 WSL2 及 Vagrant (libvirt) 格式的镜像。

## 已知问题

与其他版本一样，该版本也存在许多已知问题。 查阅 [上游清单](https://docs.redhat.com/en/documentation/red_hat_enterprise_linux/9/html/9.7_release_notes/known-issues)，了解有关这些问题的完整细节。

!!! info "废弃软件"

    Rocky Linux 9.7 存储库中一些应用程序流将不会收到任何进一步的更新。这是几乎每个版本都会发生的正常情况。请确保您使用的是最新支持的软件版本。
