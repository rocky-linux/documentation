## Note di rilascio per Rocky Linux 9.7

È possibile trovare un elenco completo della maggior parte delle modifiche nelle [note di rilascio upstream disponibili qui](https://docs.redhat.com/en/documentation/red_hat_enterprise_linux/9/html/9.7_release_notes/index).

## Aggiornare

È possibile eseguire l'aggiornamento da Rocky Linux 9.x a Rocky Linux 9.7 eseguendo `sudo dnf -y upgrade`.

!!! Note Avviso

    ```
    Rocky Linux non integra una procedura di aggiornamento da nessuna versione di Rocky Linux 8. Si consiglia di eseguire una nuova installazione del sistema operativo per eseguire l'aggiornamento a Rocky Linux 9.7.
    ```

## Immagini

Oltre alle immagini di installazione standard, sono disponibili numerose immagini, tra cui quelle per piattaforme cloud e container.

Ulteriori informazioni sugli prodotti sviluppati dal Cloud Special Interest Group e su come partecipare sono disponibili nella pagina [SIG/Cloud Wiki] (https://sig-cloud.rocky.page/).

## Installazione

Prima dell'installazione, [assicurarsi della compatibilità della CPU con questa procedura!](https://docs.rockylinux.org/gemstones/test_cpu_compat/)

Per installare Rocky Linux 9.7, visitate la [pagina di download](https://rockylinux.org/download/) e scaricate la versione corrispondente alla vostra architettura.

### RL Web Console (Cockpit)

Cockpit è un'interfaccia web per la gestione e il monitoraggio del sistema locale.

Questa nuova versione presenta uno stile dell'interfaccia utente migliorato basato su PatternFly 6.

#### Image Builder

Image Builder è ora in grado di creare immagini WSL2 e Vagrant (libvirt).
